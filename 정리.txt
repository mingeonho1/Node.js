# Node

node.js를 사용하는 이유
: 브라우저환경과 컴퓨터환경 두마리 토끼를 잡을 수 있다.
: 프론트와 백앤드 둘다 같은 언어로 구현 가능하다.
: 50퍼 이상이 node.js를 사용하고 있다.
: 많은 대기업들이 node.js를 사용하고 있다.
: 많은 커뮤니티와 툴을 가지고 있고 성능은 이미 검증을 받았다.
: 쉽게 접근가능하고 좋은 생산성을 가지고 있다.

node 의 특징
1. javascript runtime
2. single Thread
3. non-blocking i/o
4. event-driven
